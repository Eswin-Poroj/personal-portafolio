## Recomendaciones:

- **Formato**: WebP para mejor compresión (o JPG/PNG)
- **Tamaño**: Máximo 1920x1080px
- **Peso**: < 500KB por imagen
- **Optimización**: Usa herramientas como TinyPNG o Squoosh
- **Lazy Loading**: Las imágenes se cargan automáticamente con lazy loading
